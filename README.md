## 📌 About

**DriveSphere Motors** is a full-stack premium automotive web platform built with React + TypeScript. It features a stunning car showcase, AI-powered assistant, finance calculator, parts store, dealer locator, customer portal, and a complete admin CRM — all in one seamless experience.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏎️ **Car Showcase** | Browse premium vehicles with detailed specs and gallery |
| 🤖 **AI Assistant** | Intelligent chatbot to help find your perfect car |
| 💰 **Finance Calculator** | Real-time EMI & loan calculation tool |
| 🛒 **Parts Store** | Browse and order genuine spare parts |
| 📍 **Dealer Locator** | Find nearest authorized dealerships |
| 👤 **Customer Portal** | Manage bookings, test drives & service history |
| 🛠️ **Admin CRM** | Full backend dashboard for inventory & leads |
| 🌱 **Corporate CSR** | Company sustainability & social responsibility page |

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v18+

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/mawiya-47/Drive-Sphere-Motors.git
cd Drive-Sphere-Motors

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

App will run at **http://localhost:3000** 🚀
